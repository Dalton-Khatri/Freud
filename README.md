# Mental Health AI Assistant Project

## Overview
This project is a Mental Health AI Assistant designed to provide support and resources for individuals seeking help with their mental well-being. Leveraging various programming languages and technologies, this assistant aims to create a user-friendly interface and provide insightful recommendations.

## Technologies Used
- **Jupyter Notebook**: 67.1% 
- **Dart**: 15%  
- **Python**: 13.8%  
- **C++**: 2%  
- **CMake**: 1.6%  
- **Swift**: 0.2%  
- **Other**: 0.3%  

## Features
- Interactive mental health assessments.
- Personalized recommendations based on user input.
- Integration with external mental health resources.

## Getting Started
To get started with the Mental Health AI Assistant:
1. Clone the repository.
2. Install required packages.
3. Run the Jupyter Notebooks to interact with the assistant.
